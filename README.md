# 🤖 Full Stack AI Chatbot

A Full Stack AI Chatbot that delivers real-time AI-powered conversations through a modern and responsive web interface. The project features a frontend built with HTML, CSS, and JavaScript, and a backend powered by Node.js and Express.js, integrated with the OpenRouter API.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/anuragpandey8998/AI-Chatbot.git
```

### 2. Navigate to the Backend

```bash
cd AI-Chatbot/backend
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Create a `.env` File

```env
OPENROUTER_API_KEY=your_api_key_here
```

### 5. Start the Backend

```bash
npm start
```

### 6. Open the Frontend

Open `index.html` in your browser or use Live Server.
